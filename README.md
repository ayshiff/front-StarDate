# How it works ?
## Step 1
Run ```git clone https://github.com/ayshiff/front-StarDate.git ```
It will initialize the dependencies.
And then ```cd front-StarDate ```

## Step 2
Run ```npm install ```
It will initialize the dependencies.

## Step 3
Run ``` npm start ```

## Step 4
Go to ```localhost:3000```
